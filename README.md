# AI Expense Tracker

A simple expense tracking application built with **Spring Boot**, **JWT**, **PostgreSQL**, and **React (Vite)**.  
The app allows users to log in, track expenses, view monthly reports, and get basic budget insights in Indian currency (₹).

## Features
- User login with JWT authentication  
- Add and view expenses  
- Monthly expense summary  
- Savings calculation and budget advice  

## Tech Stack
- Backend: Spring Boot, Java, JWT, PostgreSQL  
- Frontend: React (Vite), CSS  

## How to Run
- Backend: `mvn spring-boot:run` → runs on `http://localhost:8080`  
- Frontend: `npm install && npm run dev` → runs on `http://localhost:5173`
